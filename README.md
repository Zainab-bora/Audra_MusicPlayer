# 🎵 Audra MusicPlayer

Audra MusicPlayer is a Full-Stack modern responsive web music player built using React and Firebase.  
It provides a smooth music streaming experience with playlist management, favorites, Google authentication, and an immersive animated UI.

---

## ✨ Features

- 🎧 Full music player with play/pause controls
- ⏭ Next / Previous track navigation
- 📱 Fully responsive mobile-friendly design
- 🌌 Animated neon background UI
- ❤️ Favorites system
- 🔍 Song search functionality
- ☁ Firebase Firestore integration
- 🔐 Google Authentication
- ➕ Add custom songs
- 🗑 Delete uploaded songs
- 🎵 Pre-installed online music library
- 📀 Animated CD-style player
- 🔊 Volume and mute controls
- ⚡ Loading skeleton animations
- 🎨 Custom branding and logo design

---
